# Novedades

Sistema para el registro y gestión de novedades de trabajadores.
Permite registrar, buscar y consultar la información de cada trabajador
junto con sus novedades asociadas.

## Tecnologías
- Java SE
- Programación Orientada a Objetos
- Streams y programación funcional

## Funcionalidades
- Registro de trabajadores
- Registro de novedades por trabajador
- Búsqueda de trabajadores por cédula
- Eliminación de trabajadores
- Resumen general de novedades

## Cómo correrlo
Clona el repositorio y ejecuta el archivo `Main.java` desde tu IDE.
```bash
git clone https://github.com/alexismoreno1211-tech/novedades-java.git
```
